# Deep Research - 深度研究

使用 Brave Search + web_fetch 做深度研究，比普通搜索更全面。

## 使用场景

- 用户要求"深度搜索"、"详细调查"、"深入了解"
- 复杂主题需要多源验证
- 需要综合多个来源的分析报告

## 工作流程

1. **第一轮**：Brave Search 广泛搜索，获取初步结果
2. **第二轮**：选择最相关的 URL，用 web_fetch 获取详细内容
3. **第三轮**：交叉验证信息，生成综合报告

## 使用方法

```bash
# 研究主题
研究: Python异步编程最佳实践

# 或者直接调用技能
使用 deep-research 技能研究 xxx 主题
```

## 输出格式

- 搜索结果摘要
- 关键发现
- 来源链接
- 进一步研究建议
