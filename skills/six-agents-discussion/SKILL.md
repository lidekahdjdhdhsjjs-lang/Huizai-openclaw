# Six Agents Discussion - 六人讨论系统

## 角色

1. **CEO (探索者)** - 战略方向、目标设定
2. **Planner (规划师)** - 任务分解、优先级
3. **Executor (执行者)** - 具体操作执行
4. **Monitor (监控者)** - 状态检查、风险预警
5. **Learner (学习者)** - 总结经验、持续改进
6. **Reporter (报告者)** - 结果汇总、对外沟通

## 使用

```
使用 six-agents-discussion 技能讨论: 你的话题
```

## 流程

1. CEO提出主题
2. Planner分解任务
3. Executor执行
4. Monitor监控
5. Learner总结
6. Reporter汇报
