# resilient-connections - 弹性连接技能

## 功能
防止传输阻塞、连接超时、网络抖动

## 核心机制

### 1. 连接池管理
- 维护多个备用连接
- 失败时自动切换
- 连接健康检查

### 2. 指数退避重试
```
重试延迟序列：1s → 2s → 4s → 8s → 16s → 32s (最大)
```

### 3. 熔断器模式
- 连续失败 5 次 → 打开熔断器
- 熔断器打开时，快速失败不重试
- 60 秒后半开尝试恢复

### 4. 超时配置
```json
{
  "connect_timeout": 5000,
  "read_timeout": 30000,
  "write_timeout": 10000
}
```

### 5. 降级策略
- 主 API 失败 → 备用 API
- 网络请求失败 → 返回缓存
- 超时 → 返回默认值
