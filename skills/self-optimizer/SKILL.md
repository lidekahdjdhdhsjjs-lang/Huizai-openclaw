# self-optimizer - 辉仔自我优化技能

## 触发条件
- 每次会话开始时
- 收到 "自优化"、"反思"、"复盘" 等关键词时

## 核心功能

### 1. 强制记忆读取
```
读取以下文件获取上下文：
- memory/YYYY-MM-DD.md (当天)
- memory/yesterday.md (昨天，如果存在)
- MEMORY.md (长期记忆)
```

### 2. 自学习报告生成
- 每日总结学到的关键知识
- 保存到 memory/self-learning-YYYY-MM-DD.md
- 标记需要改进的方向

### 3. 技能缺口识别
当遇到无法直接完成的任务时：
1. 检查现有技能是否覆盖
2. 如果没有，创建一个新技能
3. 记录到 skills/gap-log.md

### 4. 主动优化触发
每完成一个任务后，自动问自己：
- 这次有什么可以改进？
- 有什么重复操作可以自动化？
- 有什么知识应该记录下来？

## 执行流程

1. **会话启动** → 读取 memory 文件
2. **任务完成** → 生成自学习笔记
3. **定期复盘** → 识别技能缺口
4. **技能创建** → 填补能力空白
