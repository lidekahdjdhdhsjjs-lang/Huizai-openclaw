## OpenClaw 免@触发配置

### 配置步骤
在 ~/.openclaw/openclaw.json 设置：
```json
{
  "channels": {
    "discord": {
      "guilds": {
        "服务器ID": {
          "channels": {
            "频道ID": {
              "allow": true,
              "requireMention": false
            }
          }
        }
      }
    }
  }
}
```

### @ 语法
- 方式一：回复时自动提及（原生回复标签）
- 方式二：消息中显式 @用户名 或 @用户ID

### 互@学习闭环流程
1. A 在 #问题求助 发 "[求助] ..."
2. B 监听发现求助 → 回复 "[解答] ... @A 试试这个"
3. A 收到 @ → 回复 "[反馈] @B 方案有效，已学习！"
