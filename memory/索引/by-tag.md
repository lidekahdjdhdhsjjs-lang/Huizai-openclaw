# 🏷️ 标签系统

## 自动标签规则

### 按类型
| 标签 | 说明 | 文件 |
|------|------|------|
| #用户 | 用户信息 | 偏好/user-preferences.md |
| #配置 | 系统配置 | MEMORY.md |
| #任务 | 待办事项 | 待办/active.md |
| #技能 | 技能相关 | 知识库/skills.md |
| #学习 | 学习记录 | academic-learn-*.md |
| #cron | 定时任务 | 重要日期/recurring.md |

### 按状态
| 标签 | 说明 |
|------|------|
| #进行中 | 正在处理 |
| #待处理 | 等待处理 |
| #已完成 | 已完成 |
| #阻塞 | 被阻塞 |

### 按优先级
| 标签 | 说明 |
|------|------|
| #P0 | 紧急重要 |
| #P1 | 重要 |
| #P2 | 一般 |
| #P3 | 低优先级 |

### 按项目
| 标签 | 说明 |
|------|------|
| #亮仔 | 亮仔相关 |
| #辉仔 | 辉仔相关 |
| #foundry | Foundry相关 |
| #auto-company | 自动化公司 |

## 标签使用示例

```markdown
# 待办示例
- [ ] 配置Discord bot  #配置 #亮仔 #P0
- [ ] 学习新技能     #学习 #P2
- [ ] 修复cron错误  #任务 #foundry #P1
```
