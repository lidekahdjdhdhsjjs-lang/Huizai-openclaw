# Auto-Company 项目学习笔记

## 项目概述

**nicepkg/auto-company** - 全自主 AI 公司，24/7 运行
- 14 个 AI Agent，基于世界顶级专家思维模型
- 完全无人干预，自动赚钱
- 核心技术：Claude Code

## 核心特点

### 1. 团队架构 (14 Agent)

**战略层：**
- `ceo-bezos` - Jeff Bezos 商业决策
- `cto-vogels` - Werner Vogels 技术架构
- `critic-munger` - Charlie Munger 逆向质疑（唯一刹车）

**产品层：**
- `product-norman` - Don Norman 产品功能
- `ui-duarte` - Matías Duarte 视觉设计
- `interaction-cooper` - Alan Cooper 用户体验

**工程层：**
- `fullstack-dhh` - DHH 全栈开发
- `qa-bach` - James Bach 质量保证
- `devops-hightower` - Kelsey Hightower 运维部署

**商业层：**
- `marketing-godin` - Seth Godin 营销
- `operations-pg` - Paul Graham 运营
- `sales-ross` - Aaron Ross 销售
- `cfo-campbell` - Patrick Campbell 财务

**情报层：**
- `research-thompson` - Ben Thompson 市场调研

### 2. 决策原则

1. Ship > Plan > Discuss
2. 70% 信息即行动
3. 客户至上
4. 简单优先
5. 拉面盈利
6. Boring Technology
7. 单体优先

### 3. 协作流程

1. 新产品评估：research → ceo → critic → product → cto → cfo
2. 功能开发：interaction → ui → dhh → qa → devops
3. 产品发布：qa → devops → marketing → sales → operations → ceo
4. 定价变现：research → cfo → sales → critic → ceo

### 4. 安全红线

- 禁止删库
- 禁止非法活动
- 禁止泄露凭证
- 禁止 force push

### 5. 技能武器库

- deep-research (深度研究)
- web-scraping (爬虫)
- product-strategist (产品战略)
- startup-financial-modeling (财务建模)
- pre-mortem (预先验尸)
- code-review-security (代码安全审查)
- seo-content-strategist (SEO内容)
