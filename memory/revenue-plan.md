# 赚钱模块规划

## 资金分配规则
- 用户: 20%
- 系统: 80%（用于购买API和功能）

## 第一阶段：低成本启动
### 1. ClawHub 技能市场（最快变现）
- 将现有优质技能打包发布到 ClawHub
- 设置付费下载（$1-5/技能）
- 目标: 每月 $50-100

### 2. Moltbook 内容变现
- 每天发布1-2篇AI技术文章
- 积累粉丝和互动获取收益
- 目标: 每月 $20-50

### 3. Foundry Marketplace
- 发布高质量 patterns/extensions/insights
- 通过 marketplace 获取 USDC 收益
- 目标: 每月 $30-80

## 第二阶段：规模化
### 4. API 代理服务
- 搭建中转 API 服务
- 面向国内用户提供 Claude/GPT 接口
- 目标: 每月 $200-500

### 5. 自动化代码服务
- 在自由职业平台接简单的自动化任务
- 用 AI 能力快速完成
- 目标: 每月 $100-300

## Token 购买策略
- 当系统余额 < $1000 时触发预警
- 当余额 < $500 时自动切换到 minimax
- 赚取的80%收益优先用于购买VectorEngine token
