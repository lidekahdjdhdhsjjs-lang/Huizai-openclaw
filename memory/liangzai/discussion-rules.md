## 讨论流程

1. **康仔查看日期选择主题**
   - 辉仔和亮仔等待康仔选择主题
   - 根据日期/星期选择合适主题
   
2. **辉仔先发言**
   - 就主题发表观点
   - 问亮仔的想法
   
3. **亮仔继续发表**
   - 回应辉仔的观点
   - 问辉仔的想法
   - 讨论在辉仔和亮仔之间进行
   
4. **写入记忆系统**
   - 对话记录写入 memory/discussion-YYYY-MM-DD.md
   
5. **康仔总结**
   - 归纳要点
   - 形成共识

## 角色分工

- 🔵 辉仔 (探索者) - 先发言，引出话题
- 🟢 亮仔 (优化师) - 深入讨论
- 🟡 康仔 (总结者) - 主持流程 + 总结
