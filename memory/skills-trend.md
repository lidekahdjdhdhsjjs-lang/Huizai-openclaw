# GitHub 技能趋势分析

> 更新日期: 2026-02-22

## 📈 本周热门技能趋势

### 1. AI Agent 框架 (AI Agent Frameworks)
- **CowAgent** - 基于大模型的超级AI助理，支持长期记忆、任务规划
- **AstrBot** - 开源IM机器人基础设施，集成多种LLM和插件 (⭐增长中)
- **Cherry Studio** - AI生产力工作室，300+助手
- **LazyLLM** - 最简单的多Agent LLM应用构建框架
- **ApplyPilot** 🤖 NEW - AI求职Agent，自动投递简历 (314 stars, 5天)

### 2. MCP (Model Context Protocol) 相关
- **activepieces** - AI Agents & MCPs & AI工作流自动化 (~400 MCP servers)
- **CopilotKit** - 前端Agent & 生成式UI框架
- **MCP Registry** - GitHub官方MCP集成
- **agent-context-protocol** 📄 NEW - Agent Context Protocol 文档和规划系统

### 3. Computer Use Agents
- **CUA (trycua/cua)** - 开源计算机使用Agent基础设施
- **E2B** - 企业级安全沙箱环境
- **Integuru** - 首个通过逆向工程平台内部API构建无权限集成的AI Agent

### 4. Claude Code 相关生态
- **learn-claude-code** - 从0到1构建类Claude Code的纳米Agent
- **ralph-claude-code** - Claude Code自动开发循环
- **claude-code-guide** - Claude Code设置、命令、工作流、skills指南
- **AionUi** - 免费、本地、开源的24/7 Cowork和OpenClaw (for Gemini CLI, Claude Code, Codex, etc.)

### 5. 工作流与自动化
- **IntentKit** - 构建配备强大技能的开源AI Agent框架
- **zcf (Zero-Config Code Flow)** - Claude Code & Codex的零配置代码流
- **wanwu (万悟)** - 中国联通元景万悟Agent平台

### 6. 中国区热门项目
